# Backend 

Recuerda ejecutar

```
npm install
```

Validaciones de campo obligatorios
```
npm i express-validator
```

Encriptar password 
```
npm i bcryptjs
```

Generar Token
```
npm i jsonwebtoken
```
Upload File
```
npm i express-fileupload
```
Generar un identificador unico para los archivos
```
npm i uuid
```
